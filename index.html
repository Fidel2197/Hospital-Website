About Us

This project presents a modern hospital website layout with a strong first impression, service cards, appointment intake, and patient-first messaging.

The page is built with semantic HTML and organized CSS so it can be extended into a fuller hospital management system with doctor profiles, appointment storage, and authenticated patient workflows.

It is intentionally responsive, making the same experience work across desktop, tablet, and mobile screens.
