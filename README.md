# FortunatoCare Hospital Website

FortunatoCare is a responsive hospital landing page built with HTML and CSS. It presents a patient-friendly healthcare experience with appointment booking, service highlights, doctor profiles, and contact information in a clean single-page layout.

## Overview

This project was created to demonstrate front-end development for a healthcare website. The page focuses on clear navigation, accessible content, responsive sections, and a professional visual structure that could later grow into a full hospital management system.

## Features

- Responsive landing page for desktop, tablet, and mobile screens
- Appointment request form with required contact fields
- Service cards for laboratory testing, health checks, and physiotherapy
- About section explaining the purpose of the healthcare platform
- Why Choose Us section for patient trust signals
- Doctor profile cards with local image assets
- Footer navigation and contact details
- GitHub Pages-ready `index.html` entry point

## Technologies Used

- HTML5 for page structure and semantic content
- CSS3 for layout, responsiveness, and styling
- Remix Icon for healthcare and interface icons
- Local image assets for visual presentation

## Project Structure

```text
Hospital-Website/

  styles.css
  about.jpg
```

`Website.html` redirects to `index.html` so older links still work while GitHub Pages can load the project normally.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Fidel2197/Hospital-Website.git
```

Open the project folder and launch `index.html` in a browser. No build step or package installation is required.
